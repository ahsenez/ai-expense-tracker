# AI Expense Tracker

AI destekli harcama takip ve analiz uygulaması.

## Özellikler
- Harcama ekleme
- Harcama silme
- Kategori bazlı analiz
- Toplam harcama takibi
- Grafik ile veri görselleştirme
- LocalStorage ile veri kaydetme
- AI destekli harcama önerileri

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript
- Chart.js
- Render

## Proje Amacı
Kullanıcıların harcamalarını takip etmesini sağlamak ve yapay zeka destekli analiz ile finansal farkındalık oluşturmak.
